# Coffee'station ☕

A warm, inviting, and modern café website built with React, Vite, Tailwind CSS, and Framer Motion. Featuring a 3D interactive hero section using React Three Fiber.

## Features

-   **3D Hero Section**: Interactive 3D coffee cup model with rotating animations.
-   **Smooth Animations**: Scroll transitions, hover effects, and page transitions powered by Framer Motion.
-   **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
-   **Pages**:
    -   **Landing Page**: Hero, Menu Highlights, Ambience, Testimonials.
    -   **About Page**: Story and Team.
    -   **Contact Page**: Booking Form and Contact Details.
-   **Tech Stack**: React, Vite, Tailwind CSS, Three.js, React Three Fiber, Framer Motion.

## Setup Instructions

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/coffee-station.git
    cd coffee-station
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run locally**:
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` in your browser.

4.  **Build for production**:
    ```bash
    npm run build
    ```

## Development

-   **Styling**: Tailwind CSS configuration in `tailwind.config.js`. Theme colors: `coffee`, `cream`, `terracotta`.
-   **Components**: Located in `src/components`.
-   **Pages**: Located in `src/pages`.
