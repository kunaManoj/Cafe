# ‚òï Coffee'Station

A premium, modern, and highly interactive cafe website built with **React 19**, **Vite**, and **Tailwind CSS**. Designed with a "liquid glass" aesthetic, this application offers a seamless and luxurious user experience for coffee enthusiasts.


## ‚ú® Features

- **Interactive Hero Section**: A stunning landing experience with floating glass layers, parallax effects, and animated steam.
- **Dynamic Menu System**: Browse through various categories including Coffee, Tea, Desserts, Breakfast, and curated Combos.
- **Glassmorphic UI**: High-end design utilizing custom deep-glass utilities, frosted textures, and subtle micro-animations.
- **Real-time Cart Sidebar**: Manage your order effortlessly with a persistent, fluid cart experience.
- **Smooth Navigation**: Framer Motion powered page transitions and integrated Lenis smooth scrolling for a premium feel.
- **Table Reservation**: A dedicated booking system with a sleek, user-friendly interface.
- **Fully Responsive**: Optimized for every device, from mobile screens to large desktop monitors.

## üõ†Ô∏è Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kunaManoj/Cafe.git
   ```

2. Navigate to the project directory:
   ```bash
   cd coffee
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:5173`.

## üé® Design Philosophy

The project follows a **"Warm Glass"** design philosophy, combining the cozy warmth of a traditional cafe with the sleek, modern feel of glassmorphic interfaces. Key design elements include:
- **HSL-tailored coffee tones** (Creams, Caramels, and Deep Roasted Browns).
- **Layered transparency** to create depth and visual interest.
- **Physical-based animations** that react to user interaction and scroll position.
